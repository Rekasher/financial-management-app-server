# Posts App

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Create db in docker map

```bash
# up docker container
$ docker-compose up -d

# generate prisma database
$ npx prisma generate --schema=./prisma/schema


# push database to our docker container
$ npx prisma db push --schema=./prisma/schema

# down docker container
$ docker-compose down

# up docker container
$ docker-compose up -d

```
